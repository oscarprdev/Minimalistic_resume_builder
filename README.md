## Description

Minimalistic resume builder app. Create, edit, store and download your own resume in a simple way

## Stack

- Frontend
  - Typescript
  - React
  - Nextjs
  - Tailwind
- Backend
  - Cloudflare workers   
  - PostgreSQL
  - Cloudlfare R2

## Features
  - Download Resume PDF
  - Edit and store resumes
  - Choose between themes

## Work in progress
